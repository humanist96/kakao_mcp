# PlayMCP 등록용 서비스 소개문 — 왜움직여?

> 콘솔 입력 필드에 맞게 길이별로 준비. 심사 기준(창의성·편의성·안정성)을 본문에 자연스럽게 녹였다.

---

## 서비스명

**왜움직여?** — 주식 초보자를 위한 공시 통역사

## 한 줄 소개 (짧은 설명 필드용)

"내 주식이 오늘 왜 떨어졌지?" — 어려운 전자공시(DART)와 시세를 연결해 초보자의 언어로 설명하고, 상장폐지·자본잠식 같은 위험신호를 미리 경고하는 투자자 보호 AI입니다.

## 상세 소개 (긴 설명 필드용)

한국의 개인투자자는 1,400만 명이지만, 대부분 공시를 읽지 않은 채 투자합니다.
유상증자, 전환사채, 무상감자, 감사의견 거절 — 주가를 좌우하는 정보는 DART에 전부 무료로 공개되어 있지만, 전문용어의 벽에 막혀 사실상 전문가의 전유물입니다.

**왜움직여?는 이 정보 비대칭을 해소하는 통역사입니다.**

📌 **"삼성전자 오늘 왜 떨어졌어?"** 라고 물으면 —
당일 시세·거래량, 기관·외국인 수급, 최근 공시를 연결해 3줄로 설명합니다.
설명할 수 없는 움직임은 "뚜렷한 공개 요인이 없다"고 정직하게 답합니다.

🚨 **"이 종목 위험해?"** 라고 물으면 —
감사의견, 자본잠식, 관리종목, 전환사채 남발, 최대주주 잦은 변경, 횡령·배임 등
15가지 위험신호를 점검해 초보자의 말로 경고합니다.

📖 **"유상증자가 뭐야? 나한테 나쁜 거야?"** 라고 물으면 —
공시를 "무슨 일이야? / 그래서 뭐? / 나랑 무슨 상관?" 3문항으로 통역하고,
50개 투자 용어를 한 줄씩 풀어 설명합니다.

이 외에도 재무 건강진단(A~F 학점), 임원·큰손의 실제 매매 추적, 오늘의 주요 공시 요약,
"배당 4% 이상 PBR 1 이하" 같은 자연어 종목 검색을 제공합니다.

**신뢰를 위한 3가지 원칙**
1. 모든 답변에 DART 공시 원문 링크와 데이터 출처를 붙입니다 — 직접 확인할 수 있습니다.
2. 특정 종목의 매수·매도를 권유하지 않습니다. 사실의 요약과 해설만 제공합니다.
3. 데이터는 DART(금융감독원 전자공시)·KRX 시세 등 100% 공개 데이터만 사용하며, 개인정보를 수집하지 않습니다.

## 이런 걸 물어보세요 (예시 프롬프트)

- 삼성전자 오늘 왜 떨어졌어?
- 에코프로 위험한 종목이야? 점검해줘
- 카카오 최근 공시 쉽게 설명해줘
- 유상증자가 뭐야? 주주한테 나쁜 거야?
- SK하이닉스 재무 건강 상태 학점으로 알려줘
- 이 회사 임원들이 최근에 자기 주식 샀어?
- 오늘 중요한 공시 3개만 요약해줘
- 배당수익률 4% 이상이고 PBR 1 이하인 코스피 종목 찾아줘

## 제공 도구 (7종)

|---|---|
| why_moved | 종목 급등락 원인 3줄 설명 (공시+수급+시장) |
| risk_check | 위험신호 15개 룰 진단 — 투자 전 안전 점검 |
| explain_disclosure | 공시 3문항 통역 + 용어 풀이 |
| stock_health | 재무 건강진단 (5축 점수 + A~F 학점) |
| insider_signal | 내부자·5% 큰손의 공시된 실제 매매 추적 |
| daily_digest | 오늘의 주요 공시 다이제스트 |
| screen_stocks | 자연어 종목 검색 |

## 카테고리 / 태그 제안

카테고리: 금융·투자
태그: 주식, 공시, DART, 초보투자자, 투자자보호, 재무분석, 위험진단

## 유의사항 (심사 대응)

- 본 서비스의 모든 응답은 공개 데이터에 기반한 **사실의 요약이며 투자 자문·권유가 아닙니다**. 모든 응답 하단에 고지문구가 자동 포함됩니다.
- 시세는 종가 기준(지연 데이터)이며 응답에 기준일을 명시합니다.
- 매수·매도 권유 표현은 서버 차원에서 생성이 차단됩니다.

### 6번 확정 문안 (195/200자)

"내 주식이 왜 떨어졌지?" 1,400만 개인투자자의 질문에 DART 공시·시세·수급을 연결해 초보자의 말로 답하는 공시 통역사입니다. 상장폐지·자본잠식 등 15가지 위험신호를 경고해 정보 비대칭을 해소합니다. 모든 답변에 공시 원문 링크를 붙이고 권유 표현 없이 공개데이터만 쓰는 투자자 보호 서비스를 카카오톡으로 전 국민에게 전하고자 지원합니다.
